# قوالب برومبت جاهزة للنسخ

هذه مجموعة من قوالب البرومبت الجاهزة للنسخ والاستخدام المباشر مع Manus AI، مصممة لتغطية مجموعة واسعة من المهام الشائعة وتحقيق أفضل النتائج.

---

## قوالب البرمجة (Programming)

### 1. طلب كود نظيف (Clean Code)

```
أريد دالة بايثون باسم `calculate_user_age` تأخذ تاريخ الميلاد بصيغة `YYYY-MM-DD` كمدخل. **الرجاء التأكد من أن الكود يتبع مبادئ Clean Code، مع أسماء متغيرات واضحة، وتعليقات فقط عند الضرورة، وتقسيم المنطق المعقد إلى دوال مساعدة صغيرة.**
```

### 2. تطبيق نمط تصميم (Design Pattern)

```
أحتاج إلى إدارة إعدادات التطبيق في مكان واحد. **استخدم نمط التصميم Singleton لإنشاء كلاس `AppSettings` يضمن وجود نسخة واحدة فقط منه على مستوى التطبيق.**
```

### 3. مراجعة الكود (Code Review)

```
قم بمراجعة دالة البايثون التالية. هل هناك أي مشاكل تتعلق بالأداء أو قابلية القراءة؟ هل تتبع إرشادات PEP 8؟

[الصق الكود هنا]
```

### 4. تنقيح الأخطاء (Debugging)

```
أحصل على الخطأ `TypeError: 'NoneType' object is not iterable` في هذا الكود. اشرح لي سبب هذا الخطأ واقترح كيفية إصلاحه.

[الصق الكود ورسالة الخطأ هنا]
```

### 5. إعادة الهيكلة (Refactoring)

```
هذه الدالة أصبحت طويلة جدًا ومعقدة. قم بإعادة هيكلتها عن طريق تقسيمها إلى دوال أصغر وأكثر تركيزًا، مع تحسين أسماء المتغيرات لجعلها أكثر وضوحًا.

```

## قوالب الدقة والبحث (Accuracy & Research)

### 1. التحقق من صحة معلومة

```
تحقق من صحة الادعاء التالي: 'شرب ثمانية أكواب من الماء يومياً ضروري للصحة'. استخدم أداة `search` مع نوع `research` للبحث عن دراسات علمية حديثة حول هذا الموضوع. قم بتلخيص النتائج مع ذكر المصادر التي استندت إليها باستخدام `browser` لقراءة محتواها بالكامل. إذا كانت هناك آراء متضاربة، فوضحها بالتفصيل.
```

### 2. تقسيم مهمة معقدة

```
أريد كتابة مقال مفصل حول 'تأثير الذكاء الاصطناعي على سوق العمل بحلول عام 2030'. استخدم أداة `plan` لتقسيم هذه المهمة إلى مراحل بحث وكتابة. يجب أن تشمل الخطة البحث عن تقارير اقتصادية، وتحليل أنواع الوظائف المتأثرة، وصياغة هيكل المقال. ابدأ بتنفيذ المرحلة الأولى من الخطة.
```

### 3. الحصول على بيانات محددة وموثقة

```
ابحث عن أحدث البيانات المتاحة حول حجم سوق التجارة الإلكترونية في منطقة الشرق الأوسط وشمال أفريقيا. استخدم `search(type='data', queries=['e-commerce market size MENA report'])` و `search(type='research', queries=['e-commerce growth Middle East academic'])`. قم بزيارة أفضل 3 نتائج باستخدام `browser` واستخرج الأرقام الرئيسية والإحصائيات. قدم لي النتائج في جدول Markdown مع روابط مباشرة للمصادر.
```
